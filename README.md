Annual Report 2025

This repository contains R scripts and analytical workflows developed for the Annual Report 2025, focusing on fisheries data analysis and visualization.

Contents

Data processing and cleaning scripts

Catch composition analysis

Year wise and month wise landing trends

Comparative fishery assessments

Generation of publication ready figures and tables

Tools Used

R

ggplot2

dplyr

tidyr

openxlsx

patchwork

Purpose

The repository ensures reproducibility and transparency in fisheries data analysis. It serves as a reference for methods used in compiling the Annual Report 2025 and can be adapted for future assessments and reporting exercises.
